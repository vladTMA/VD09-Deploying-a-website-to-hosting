## 📌 О проекте
«Кликер» — это учебный проект, демонстрирующий:
- работу с Flask и SQLAlchemy  
- регистрацию и авторизацию пользователей  
- хеширование паролей  
- защищённые маршруты  
- шаблоны Jinja2  
- кастомный игровой UI  
- деплой на PythonAnywhere
